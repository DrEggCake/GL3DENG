# Learning OpenGL

This repo contains my code and notes I took while learning OpenGL from [LearnOpenGL](https://learnopengl.com/),
implemented in Java using [LWJGL](https://www.lwjgl.org/).

----

## About

I'm working through LearnOpenGL book and re creating examples using LWJGL instead of C++.

Mainly for:
- experiment and practice
- translating examples from C++ to Java
- tracking my progress

Structure is currently minimal as I'm still building a understanding of the concepts.
